## Daily Workflow (After Initial Setup)

### Every time you work in Figma Make:

1. ☐ Make changes in Figma Make
2. ☐ Copy changed files to local GitHub folder (overwrite old versions)
3. ☐ Open GitHub Desktop (it auto-detects changes)
4. ☐ Write commit message describing what changed
5. ☐ Click "Commit to main"
6. ☐ Click "Push origin"

**Time: 3-5 minutes**

---

## Emergency: Restore Previous Version

If something breaks:

1. ☐ Open GitHub Desktop
2. ☐ Click "History" tab
3. ☐ Find the working version
4. ☐ Right-click that commit
5. ☐ Select "Revert changes"
6. ☐ Push the revert
7. ☐ Copy restored files back to Figma Make
