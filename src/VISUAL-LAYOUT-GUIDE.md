```
1. H1 (28px) - Page title
2. H2 (24px) - Section headers
3. Price (24px-36px) - Pricing
4. H3 (20px) - Product titles
5. Body (16px) - Main content
6. Small (14px) - Secondary info
7. Tiny (12px) - Labels
```

### Weight Scale
```
1. Bold (700) - Headings, emphasis
2. Semibold (600) - Sub-headings
3. Medium (500) - Buttons
4. Regular (400) - Body text
5. Light (300) - De-emphasized
```

### Color Hierarchy
```
1. #003b3c - Primary text (headings)
2. #666666 - Secondary text (body)
3. #999999 - Tertiary text (labels)
4. #009296 - Interactive (links, buttons)
5. #e8e4d8 - Borders, dividers
```
